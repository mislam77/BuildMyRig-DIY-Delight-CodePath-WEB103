About this web app: **BuildMyRig is a web application that allows users to customize and build their own PC by selecting various components such as CPU, GPU, motherboard, RAM, storage, PSU, cooling, and OS. Users can add parts to their cart, create custom builds, and manage their builds by updating or deleting parts. The application also includes validation to prevent users from adding multiple parts of the same type to a build, with error notifications displayed via toast messages.**

**Tech Stack - React, Typescript, TailwindCSS, Nodejs, Express, PostgreSQL**

Time spent: **6** hours
